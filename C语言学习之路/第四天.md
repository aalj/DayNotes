### 模块化变成

1. 在写程序的时候 自定义的函数 如果是放于`main` 后面在程序运行的时候是或出错的。因为先执行的main函数，并不知道你的函数在什么地方定义（简单理解），所以这里用到的`<函数的声明>`

   1. 在main函数之前对函数进行声明

   2. ```c
      #includ<stdio.h>
      //main函数之前声明函数
      //这句可以理解没函数的声明但是没有函数体；
      void   demo(int a);
      void main(){
        //调用自己定义函数
        int index=0;
        demo(index);
      }
      //函数的定义在这里
      void demo(int a){}
      ```

   3. > c语言中最好养成 先定义函数，在实现函数的习惯
