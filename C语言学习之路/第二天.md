1. > `if...else`流程控制语句

   ```c
   //简单的if 语句
   if（/*条件*/）
     {
       //条件为真执行这里的语句
     }
   //完整的if..else语句
   if(/*条件*/)
     {
       //条件为真执行这里的语句
     }
   else
     {
       //条件为假执行这里的语句
     }
   //复杂的的语句， else..if和else集合使用
   if()
     {
       
     }
   else if()
     {
       
     }
   else if()
     {
       
     }
   else 
     {
       
     }
   ```

   注意： 在 `java` `c++` 等一些语言中 定义了`boolean` 类型  `true` `false`  但是在c语言中没有定义这种类型，所以在c语言中，把`0`作为假 把`非0`作为真
